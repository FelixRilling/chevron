Bootstrapping which immediately returns the initializer. This is useful for injectables which do not require any other initialization. Note that by using this bootstrapping, no usage of dependencies for this value is possible.

Injectable(instance, dependencies, options)

Source:
  • chevron.common.js, line 319

Registers a new injectable on a container. See Chevron#registerInjectable for details.

Decorator function for use with TypeScript. Use this decorator on a variable or function/class expression.

Parameters:
Name Type Description
instance

Chevron instance to register the injectable on.
dependencies

Definitions of this injectables dependencies. See Chevron#registerInjectable for details.
options

Options for this injectable. See Chevron#registerInjectable for details.
Throws:

  • Error when an injectable with the requested name is already registered.

  • TypeError when no name can be determined for this injectable or any of its dependencies.
