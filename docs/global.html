Autowired(instance, name, context)

Source:
  • chevron.common.js, line 291

Retrieves an instantiated injectable, recursively instantiating dependencies if they were not instantiated before, and sets the value on the field/property for this decorator. See Chevron#getInjectableInstance for details.

Decorator function for use with TypeScript. Use this decorator on a object property or class field.

Parameters:
Name Type Default Description
instance

Chevron instance to retrieve the injectable from.
name

Either a raw string name or a nameable value that should be retrieved. See Chevron#registerInjectable for details.
context null

Context to be used for instance checks. See Scope for details.
Throws:

  • TypeError when no name can be determined for the provided nameable.

  • Error when the injectable or a dependency cannot be found.

  • Error when recursive dependencies are detected.

Injectable(instance, dependencies, options)

Source:
  • chevron.common.js, line 272

Registers a new injectable on a container. See Chevron#registerInjectable for details.

Decorator function for use with TypeScript. Use this decorator on a variable or function/class expression.

Parameters:
Name Type Description
instance

Chevron instance to register the injectable on.
dependencies

Definitions of this injectables dependencies. See Chevron#registerInjectable for details.
options

Options for this injectable. See Chevron#registerInjectable for details.
Throws:

  • Error when an injectable with the requested name is already registered.

  • TypeError when no name can be determined for this injectable or any of its dependencies.
