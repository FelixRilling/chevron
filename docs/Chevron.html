registerInjectable(initializer, options)

Source:
  • chevron.common.js, line 200

Registers a new injectable on this container.

Parameters:
Name Type Description
initializer

Initial value of this injectable. This can be any value, but usually a class or a different kind of function. During retrieval, the initial value might be transformed by the bootstrapper (see Bootstrapping for details). If no name is provided in the options (see description of the options parameter, section "name"), a name will be determined from the initializer through getName. or a value which is nameable. For details on nameable values see getName.
options

Options for this injectable. The following options exist:

  • name: Name for this injectable. If this is not provided, the name will be determined based on the initializer. (see description of the initializer parameter)
  • bootstrapping: Bootstrapping strategy to use when instantiating this injectable (see Bootstrapping for details). By default, DefaultBootstrappings.IDENTITY is used. If your injectable is a class or factory function, consider using DefaultBootstrappings.CLASS or DefaultBootstrappings.FUNCTION instead respectively, or provide your own.
  • scope: Scoping strategy to use when retrieving instances (see Scope for details). By default, DefaultScopes.SINGLETON is used. For different use cases, see DefaultScopes.PROTOTYPE or provide your own.

Throws:

  • Error when an injectable with the requested name is already registered.

  • TypeError when no name can be determined for this injectable or any of its dependencies.
